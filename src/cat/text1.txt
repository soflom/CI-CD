

INTRODUCTION



	In this project you will learn and develop basic Bash utilities for working with C programming language texts. 
These utilities (cat and grep) are often used in the Linux terminal. As part of the project you'll learn the 
organization of the Bash utilities and solidify knowledge of structured programming.
